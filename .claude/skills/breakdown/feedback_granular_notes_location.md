---
name: Granular notes belong inside the book folder
description: concepts/, facts/, procedures/, quotes/ go inside the book root, not at vault root
type: feedback
---

Granular note folders (`concepts/`, `facts/`, `procedures/`, `quotes/`) must be created **inside the book root folder** (e.g. `summaries/the-tyranny-of-merit/concepts/`), not at the vault root level.

**Why:** The vault skill diagram shows them as children of `<book-root>/`, scoped to that book.

**How to apply:** When writing granular notes, always prefix the path with the book root (e.g. `summaries/<book-name>/concepts/`).
