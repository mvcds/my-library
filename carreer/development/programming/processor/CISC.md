#concept

An [[Instruction Set Architecture]] where a single instruction packs an arbitrary number of operations, causing their size to vary, which makes it more [[Complex]] because it has to know how to process them which uses more [[energy|power]]

Optimized for peak [[Performance]]; trades power [[efficiency]] and heat for computational [[carreer/development/Throughput|Throughput]]. Dominates PCs, laptops and [[Server|Servers]]
